### 关键 schema 不匹配（阻塞 #1 的具体细节）

`FluxPairedDatasetV2.__getitem__`（`uno/dataset/uno.py:76`）期望每条样本有：

```json
{"image_path": "...",   // 或 "image_paths": [...]
 "prompt":     "...",
 "image_tgt_path": "..."}
```

但 `uno_1m_total_labels.json` 实际是：

```json
{"img_path1":     "split1/...png",
 "img_path2":     "split1/...png",
 "caption":       {"img_path1": "...", "img_path2": "...", "judgment": "...", "subject": [...]},
 "vlm_filter_cot": {...}}
```

也就是说原 JSON **每个样本有 2 张图（视角不同但 subject 相同）** + 共享 caption。需要做一次转换：
- `image_paths = [img_path1]`（ref）— 或者两个都当 ref 喂
- `image_tgt_path = img_path2`（target）
- `prompt = caption.img_path1` 或 `caption.img_path2`（两者等价）

### 建议处理顺序

**必须先做**（否则跑不起来）：
1. 生成 `uno_1m_total_labels_convert.json`（一次性数据预处理，不算改代码）
2. 设 `WANDB_MODE=disabled` 环境变量（或 `pip install wandb`）

**建议同时做**：
3. `accelerate config --compute_environment=...` 做一次交互式配置（不致命，但 8 卡 deepspeed multi-node 设置值得调）
